# Body Matter

PWA Next.js: rutinas, sesión con temporizador, historial, calendario y medidas. Comparte proyecto Firebase con [Eaty](https://github.com/RandyGrullon/Eaty).

## Desarrollo

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

Puerto por defecto: **3001** (ver `package.json`).
