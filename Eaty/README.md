# Eaty

App Next.js (App Router) + Firebase + Groq para registro y análisis de comidas.

## Desarrollo

1. `pnpm install`
2. Copia [`.env.example`](.env.example) a `.env.local` y completa las variables.
3. `pnpm dev`
